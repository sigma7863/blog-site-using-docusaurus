### 中国のログイン文化
	なぜ中国のログイン方法はメールアドレスではなく電話番号やQRコードが主流なの？
	
	中国のログイン文化の背景を、利用環境・サービス設計・本人確認の3つに分けて見ます。必要なら最後に、日本との違いも一緒に整理します。
	
	思考時間: 24s
	
	大きい理由は、**中国のネット利用がほぼ完全にスマホ中心**だからです。CNNICの報告では、2024年6月時点でインターネット利用者の**99.7%がモバイル経由**でアクセスしています。だからログインや本人確認も、メールより「スマホを前提にした導線」のほうが自然になります。
	
	次に、**電話番号が“連絡先”以上のIDとして扱われやすい**ことです。中国ではSIMの開通に実名登録が必要で、政府系の案内でもその点が明記されています。さらに中国の法令でも、電話利用者の実名登録制度を徹底するよう求めています。つまり電話番号は、メールより「本人にひもづきやすい」「復旧に使いやすい」認証軸になりやすいです。
	
	QRコードが多いのは、**スマホ同士・PCとスマホをつなぐのに便利で、すでに定着しているから**です。Tencent系の文書でも、初回ログインでWeChatのスキャンコードログインを勧め、その後に電話番号の確認と紐づけを行う流れが示されています。つまり中国では、電話番号が「本人確認の芯」、QRコードが「端末間の受け渡しや承認の入口」として使われやすい、という構図です。メールは使われますが、一般に複数作りやすく放置もしや　すいので、IDの中心にはなりにくいです。
	
	必要なら次に、日本・中国・米国のログイン文化の違いを1枚で比べてまとめます。

### 中国語の言語タグ(zn, 簡体字, 繁体字など)についてAI聞いたこと
	
						< ここまでのやり取りをまとめて
	
	OK、ここまでをスッキリ整理してまとめるね。
	
	# 全体像（これが一番大事）
	
	中国語のタグはこの3つの組み合わせ：
	
	**① 言語（zh） + ② 文字（Hans / Hant） + ③ 地域（CN / TW など）**
	
	# 1. `zh` の意味（言語）
	
	- `zh` = 中国語
	- 中国語の「中文（Zhōngwén）」の **Zh** が由来
	- 規格は ISO の ISO 639
	
	👉 つまり「中国語という大きなカテゴリ」
	
	---
	
	# 2. 簡体字と繁体字（文字の違い）
	
	## 簡体字（Hans）
	
	- 画数を減らした漢字
	- 主に：
	    - 中国本土
	    - シンガポール
	
	例：
	
	- 國 → 国
	- 學 → 学
	
	👉 タグ：`zh-Hans`
	
	---
	
	## 繁体字（Hant）
	
	- 昔ながらの漢字
	- 主に：
	    - 台湾
	    - 香港
	    - マカオ
	
	👉 タグ：`zh-Hant`
	
	---
	
	# 3. 地域コード（どこ向けか）
	
	- `zh-CN` = 中国本土
	- `zh-TW` = 台湾
	- `zh-HK` = 香港
	- `zh-MO` = マカオ
	
	👉 これは「どの地域向けの中国語か」
	
	---
	
	# 4. 一番混乱するポイント（重要）
	
	## 文字 vs 地域は別物
	
	- `zh-CN` → 中国向け（地域）
	- `zh-Hans` → 簡体字（文字）
	
	つまり：
	
	|タグ|意味|
	|---|---|
	|zh-CN|中国向け|
	|zh-TW|台湾向け|
	|zh-Hans|簡体字|
	|zh-Hant|繁体字|
	
	👉 **地域 ≠ 文字**
	
	---
	
	# 5. よくある組み合わせ
	
	より正確にするとこうなる：
	
	- `zh-Hans-CN` → 中国本土 + 簡体字
	- `zh-Hant-TW` → 台湾 + 繁体字
	- `zh-Hant-HK` → 香港 + 繁体字
	
	---
	
	# 6. 実務での使い分け（重要）
	
	## シンプルにやるなら
	
	- 簡体字 → `zh-Hans`
	- 繁体字 → `zh-Hant`
	
	## 地域も分けるなら
	
	- 中国向け → `zh-CN`
	- 台湾向け → `zh-TW`
	
	---
	
	# 7. 超短縮まとめ
	
	- `zh` = 中国語
	- `Hans` = 簡体字
	- `Hant` = 繁体字
	- `CN` = 中国
	- `TW` = 台湾
	
	👉 **「言語 + 文字 + 地域」で構成される**
	
	---
	
	# 最後にイメージ
	
	zh（中国語）  
	 ├─ zh-Hans（簡体字）  
	 │    └─ zh-Hans-CN（中国本土）  
	 └─ zh-Hant（繁体字）  
	      ├─ zh-Hant-TW（台湾）  
	      └─ zh-Hant-HK（香港）
	
	---
	
	ここまで理解できれば、Web・i18n・翻訳周りではほぼ困らなくなるレベル。
